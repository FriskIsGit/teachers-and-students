## Spring Boot ORM implementation based on students and teachers to showcase ManyToMany relationship

### Technologies in use:
- Maven
- Spring Boot JPA
- MySQL

### Model is made up of 4 layers:
1. Repositories
2. DAOs
3. Services
4. Controller

### There are tests for DAOs, Services and the Controller
